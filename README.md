# LLM_for_time_series

1. Демонстрация: в ноутбуке `notebooks/00_demo.ipynb`
2. Код: в `src/`

## Краткое описание

Код для работы со временными рядами. По характеристикам временного ряда и его описанию выбирает модель для предсказания и способ визуализации.

Файл `src/main_pipeline.py` --- основной файл.

## Принцип работы:

**Шаг 1** извлечение признаков

В файле `src/feature_extractor.py` из временного ряда извлекаются признаки.

**Шаг 2** LLM-report

Извлеченные признаки, описание временного ряда подаются в LLM для генерации отчета с общими выводами по временному ряду и рекомендациями по его обработке и визуализации.

См. файл `src/llm_stage_1_reporter.py`

**Шаг 3** LLM-codegen

По временному ряду, его признакам, отчету генерируется код для предсказания временного ряда, вычисляются метрики, все визуализируется.

См. файл `src/llm_stage_2_codegen.py`

**Замечание** При создании проекта использовались нейросети.
